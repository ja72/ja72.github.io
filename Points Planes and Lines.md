  <div class="stackedit__html"><h1 id="geometry-with-homogeneous-coordinates">Geometry with Homogeneous Coordinates</h1>
<p>A general feature of homogeneous coordinates is that non-zero scalar multiples of the coordinates do not change the description of the underlying geometry. If <span class="katex--inline"><span class="katex"><span class="katex-mathml"><math><semantics><mrow><mi mathvariant="bold-italic">P</mi></mrow><annotation encoding="application/x-tex">\boldsymbol{P}</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height: 0.68611em; vertical-align: 0em;"></span><span class="mord"><span class="mord"><span class="mord boldsymbol" style="margin-right: 0.15972em;">P</span></span></span></span></span></span></span> is a point, then so is <span class="katex--inline"><span class="katex"><span class="katex-mathml"><math><semantics><mrow><mn>2</mn><mi mathvariant="bold-italic">P</mi></mrow><annotation encoding="application/x-tex">2 \boldsymbol{P}</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height: 0.68611em; vertical-align: 0em;"></span><span class="mord">2</span><span class="mord"><span class="mord"><span class="mord boldsymbol" style="margin-right: 0.15972em;">P</span></span></span></span></span></span></span> the exact same point in space.</p>
<p>Another feature is the ability to describe geometry not only in finite space, but also on the horizon at infinity (more like an infinite sphere).</p>
<h2 id="points">1. Points</h2>
<p>Points have one vector and one scalar component (4 quantities) in general. The vector component is a scalar multuple of the location of the point.</p>
